#### `settings/settings.json`
This report contains various settings specific to your pump.
##### Setup code
`openaps report add settings/settings.json JSON pump read_settings`
##### Sample contents
`{
  "low_reservoir_warn_point": 20, 
  "keypad_lock_status": 0, 
  "maxBasal": 6.0, 
  "temp_basal": {
    "percent": 100,

  },
